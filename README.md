mongo_tutorial
==============


## Installation

```
git clone git@github.com:kjirou/mongo_tutorial.git
cd mongo_tutorial
npm install
```


## Use Sample Database Generator

Usage:

```
path/to/mongo_tutorial/bin/gendb [-d db-name] generator-script-name
```

For example:

```
path/to/mongo_tutorial/bin/gendb blog
```
